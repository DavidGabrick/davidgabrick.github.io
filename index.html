Autonomous Turret Tracker (WIP)

A 2-axis pan-tilt vision tracking system driven by an MG996R servo on the pan axis and an SG90 on the tilt axis, mounted on a custom-designed 3D-printed PLA bracket modeled in Fusion 360. A computer vision pipeline written in Python using OpenCV performs real-time detection of human faces and moving objects such as drones from a USB webcam feed, computing centroid error relative to the frame center. A proportional controller maps the pixel error on both axes to servo angle corrections, which are transmitted wirelessly to the ESP32-WROOM-32 (dual-core 240 MHz) over Wi-Fi. Firmware written in C++ using the Arduino framework and ESP32Servo library receives the joint angle targets and drives both axes via PWM. Servo power lines are conditioned with 100 µF decoupling capacitors and Schottky flyback diodes to suppress transient noise from the ESP32 rail.
